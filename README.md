# 🍰 Pantry to Oven

Pantry to Oven is a responsive frontend web application that helps users discover baking recipes, learn new techniques, and follow step-by-step instructions. The project provides a clean and user-friendly interface for exploring recipes and improving baking skills.

## ✨ Features

- 🔍 Search recipes by name
- 🏷️ Filter recipes by category and difficulty
- ❤️ Add recipes to Favorites
- 🕒 Recently Viewed recipes
- 📖 Detailed recipe pages
- ⏱️ Step-by-step baking guide with timers
- 👤 User Login & Profile (Local Storage)
- 📱 Fully Responsive Design
- 🎨 Modern and clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
- JSON


## 💡 Key Functionalities

- Browse baking recipes
- Search and filter recipes
- Save favorite recipes
- View recently opened recipes
- Step-by-step recipe instructions
- User profile stored using Local Storage
